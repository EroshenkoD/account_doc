Сайт для просмотра и скачивания загруженых файлов расположен по ссылке: https://accountdocument.herokuapp.com/look_and_download_dd. Программа была предусмотрена только для работы по локальной сети, но облачная PaaS-платформа Heroku хранит файлы, загруженые не через git, только в течении 24-х часов, поэтому был добавлен статический PDF файл в проект и во время просмотра файлов загружается только он. 
	
	Краткое описание продукта
	
 Программа позволяет вести учет КД (конструкторской документации), а именно:
 	- Вести учет всех коррекций документа КД.
 	- Позволяет выполнять удобный поиск и просмотр КД, загрузку его исходников.
 	- Возможность хранить документ КД в различных форматах, например, окончательным документом детали есть чертеж с подписями,
 	  он отсканировани и хранится в формате .pdf, но создан он в программе AutoCAD и исходником есть формат .dwg, паралельно 
 	  могут храниться другие файлы в различных форматах. Форматы добавляются через панель админа.
 	- Защита от опечаток и дублирования документации.
 	- Визуальное выдиление документа в зависимости от статуса, расширить который можно с панели администратора.
 	- Файлы с расширением .pdf автоматически открываются в браузере для просмотра, остальные автоматически скачиваются.
